## SfChart

### Bug fixes
{:#sfchart-bug-fixes}

* \#217316 – Chart axis scroll bar thumb is visible with optimum length when very small [`ZoomFactor`](http://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfChart.WPF~Syncfusion.UI.Xaml.Charts.ChartAxisBase2D~ZoomFactor.html) value is set to axis.

* \#218242 – Minor grid lines of the axis will not overlap with the major gridlines of the axis.

* \#219168 – The `ArgumentOutOfRangeException` will no longer be thrown when zooming in y-axis plot offset area.

### Features
{:#sfchart-features}

* The stacked doughnut support has been provided to chart.

* Now, the chart control supports [`RangeColumnSeries`](http://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfChart.WPF~Syncfusion.UI.Xaml.Charts.RangeColumnSeries.html) chart type with a single y-value.

* Now, the data label and [`PieSeries`](http://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfChart.WPF~Syncfusion.UI.Xaml.Charts.PieSeries.html)/[`DoughnutSeries`](http://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfChart.WPF~Syncfusion.UI.Xaml.Charts.DoughnutSeries.html) segments can be connected with horizontal straight line.

* Support has been provided to group the data points into `Others` category in [`PieSeries`](http://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfChart.WPF~Syncfusion.UI.Xaml.Charts.PieSeries.html) and [`DoughnutSeries`](http://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfChart.WPF~Syncfusion.UI.Xaml.Charts.DoughnutSeries.html) charts.

* The text or any view can be added to the center of the doughnut chart.

### Enhancements
{:#sfchart-enhancements}

* Now, the dashes can be applied to the polar and radar charts when rendering as lines.
