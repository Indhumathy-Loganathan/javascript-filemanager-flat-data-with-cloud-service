## Gantt

### Bug fixes

* \#326268 – An argument exception will no longer be thrown when inserting a single task dynamically into an empty gantt.

* \#164557 – The background panel will now be refreshed when changing the [`ScheduleType`](https://help.syncfusion.com/cr/wpf/Syncfusion.Windows.Controls.Gantt.GanttControl.html#Syncfusion_Windows_Controls_Gantt_GanttControl_ScheduleType) dynamically to numeric.

* \#163751 - GanttSchedule will now be stretched to fill the entire available space when the control size is changed dynamically.
