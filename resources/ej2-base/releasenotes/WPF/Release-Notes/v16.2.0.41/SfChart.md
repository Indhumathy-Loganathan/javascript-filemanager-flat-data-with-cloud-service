## SfChart

*  \#199872 – Axis label will no longer be shifted while resizing the chart to minimum size with [`LabelRotationAngle`](http://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfChart.WPF~Syncfusion.UI.Xaml.Charts.ChartAxis~LabelRotationAngle.html#).
