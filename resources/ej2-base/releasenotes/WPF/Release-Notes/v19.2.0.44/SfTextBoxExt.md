## SfTextBoxExt

### Features
{:#sftextboxext-features}

* \#I320684 - Now, [SfTextBoxExt](https://help.syncfusion.com/cr/wpf/Syncfusion.Windows.Controls.Input.SfTextBoxExt.html) supports clear button for single selection mode.

* \#I320677 - Now, [SfTextBoxExt](https://help.syncfusion.com/cr/wpf/Syncfusion.Windows.Controls.Input.SfTextBoxExt.html) supports  highlighting the unmatched characters in the suggestion box's filtered items  by setting [TextHighlightMode](https://help.syncfusion.com/cr/wpf/Syncfusion.Windows.Controls.Input.SfTextBoxExt.html#Syncfusion_Windows_Controls_Input_SfTextBoxExt_TextHighlightMode) to `unmatched`, as well as styling support for highlighted items in the suggestion box.

* \#I320396 - Now, the number of items displayed in the suggestion box can be customized.

* \#I320680 - Now, [SfTextBoxExt](https://help.syncfusion.com/cr/wpf/Syncfusion.Windows.Controls.Input.SfTextBoxExt.html) supports highlighting background color of  the [selecteditem](https://help.syncfusion.com/cr/wpf/Syncfusion.Windows.Controls.Input.SfTextBoxExt.html#Syncfusion_Windows_Controls_Input_SfTextBoxExt_SelectedItem) in the suggestion box.
