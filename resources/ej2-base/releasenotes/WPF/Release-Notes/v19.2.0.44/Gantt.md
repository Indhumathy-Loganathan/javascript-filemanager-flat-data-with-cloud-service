## Gantt

### Bug fixes

* \#I326268 – An argument exception will no longer be thrown when inserting a single task dynamically into an empty gantt.

* \#F164557 – The background panel will now be refreshed when changing the [`ScheduleType`](https://help.syncfusion.com/cr/wpf/Syncfusion.Windows.Controls.Gantt.GanttControl.html#Syncfusion_Windows_Controls_Gantt_GanttControl_ScheduleType) dynamically to numeric.

* \#F163751 - Gantt schedule will now be stretched to fill the entire available space when changing the control size dynamically.

* \#F165936 - Task will be updated by pixels while dragging or resizing, after disabling the [`ShowHolidays`](https://help.syncfusion.com/cr/wpf/Syncfusion.Windows.Controls.Gantt.GanttControl.html#Syncfusion_Windows_Controls_Gantt_GanttControl_ShowHolidays).
