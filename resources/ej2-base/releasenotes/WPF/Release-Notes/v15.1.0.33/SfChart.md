## SfChart

### Bug Fixes
{:#sfchart-bug-fixes}


*  \#167455 - Designer exception will no longer be thrown while using [`SfDateTimeRangeNavigator`](https://help.syncfusion.com/wpf/sfdatetimerangenavigator/overview) inside the Expander control. 

*  \#168884 - Now, [`Trackball`](https://help.syncfusion.com/wpf/sfchart/interactive-features#trackball) axis label will update properly with real time data. 

*  \#168533, \#170385 - Now,[`ChartToolTip`](https://help.syncfusion.com/wpf/sfchart/interactive-features#tooltip) XData property will update properly in [`LineSeries`](https://help.syncfusion.com/wpf/sfchart/series#line).

*  \#167981 –  Real time update performance has been improved in [`FastLineSeries`]( https://help.syncfusion.com/wpf/sfchart/series#fast-line-bitmap).

*  \#169181- Now, [`Legend`]( https://help.syncfusion.com/wpf/sfchart/legend)  will be placed properly while setting [`LegendPosition`]( https://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfChart.WPF~Syncfusion.UI.Xaml.Charts.ChartLegend~LegendPosition.html)  as Inside.

*  \# 170796 –Now, the last axis label in [`DateTimeAxis`](https://help.syncfusion.com/wpf/sfchart/axis#datetimeaxis) will not be rendered when its position exceeds [`Maximum`]( https://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfChart.WPF~Syncfusion.UI.Xaml.Charts.DateTimeAxis~Maximum.html) value.


### Features
{:#sfchart-features}


*  Provided Direction support for [`ErrorBarSeries`]( https://help.syncfusion.com/wpf/sfchart/errorbar).

*  New BoxAndWhiskerSeries type has been added.

*  Provided localization support for [`SfChart`]( https://help.syncfusion.com/wpf/sfchart/getting-started).

*  Provided [`trackball`]( https://help.syncfusion.com/wpf/sfchart/interactive-features#trackball) support for [`TechnicalIndicators`]( https://help.syncfusion.com/wpf/sfchart/technical-indicators).

*  Provided support for displaying grouped [`trackball`]( https://help.syncfusion.com/wpf/sfchart/interactive-features#trackball) label in multiple series when [`LabelDisplayMode`](https://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfChart.WPF~Syncfusion.UI.Xaml.Charts.ChartTrackBallBehavior~LabelDisplayMode.html) is Group. 


### Breaking Changes
{:#sfchart-breaking-changes}


*  Default [`trackball`]( https://help.syncfusion.com/wpf/sfchart/interactive-features#trackball) tooltip appearance has been changed for [`Financial series`](https://help.syncfusion.com/uwp/sfchart/series#financial-charts).
