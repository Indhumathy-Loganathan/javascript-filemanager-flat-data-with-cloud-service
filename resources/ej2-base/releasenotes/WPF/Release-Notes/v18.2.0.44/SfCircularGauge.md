## SfCircularGauge

### Features
{:#sfcirculargauge-features}

* \#50187 – Circular gauge [`Ranges`](https://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfGauge.WPF~Syncfusion.UI.Xaml.Gauges.CircularRange.html) display gradient effect with smooth color transition.
* \#51624 – Interactively [`Pointer`](https://help.syncfusion.com/cr/cref_files/wpf/Syncfusion.SfGauge.WPF~Syncfusion.UI.Xaml.Gauges.CircularPointer.html) position can be changed by dragging it to the desired location. Applicable for needle pointer and symbol pointer.
