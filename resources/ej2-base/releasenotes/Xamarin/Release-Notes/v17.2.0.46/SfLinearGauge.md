## SfLinearGauge

### Bug fixes

* \#240706 - [`SymbolPointer`](https://help.syncfusion.com/xamarin/circulargauge/pointers#marker-pointer) is rendered at the maximum scale value when setting the pointer value greater than maximum value.

* \#241573 - The [`ScaleOffset`](https://help.syncfusion.com/cr/cref_files/xamarin/Syncfusion.SfGauge.XForms~Syncfusion.SfGauge.XForms.LinearScale~Offset.html) value is considered for [`Annotation`](https://help.syncfusion.com/cr/cref_files/xamarin/Syncfusion.SfGauge.XForms~Syncfusion.SfGauge.XForms.SfLinearGauge~Annotations.html) rendering.

* \#242461 - Exception no longer be thrown when setting annotation's ScaleValue and the minimum and maximum values of scale to zero.
