## SfNumericUpDown

### Features
{:#sfnumericupdown-features}

* #I419024 - [UWP] The [`ValueChanging`](https://help.syncfusion.com/cr/xamarin/Syncfusion.SfNumericUpDown.XForms.SfNumericUpDown.html#Syncfusion_SfNumericUpDown_XForms_SfNumericUpDown_ValueChanging) event will not be triggered multiple times when single tapping the increment or decrement button after page navigation.
* #I417103 - [UWP] The correct user action value will be obtained in the [`ValueChanging`](https://help.syncfusion.com/cr/xamarin/Syncfusion.SfNumericUpDown.XForms.SfNumericUpDown.html#Syncfusion_SfNumericUpDown_XForms_SfNumericUpDown_ValueChanging) event arguments when [`SpinValidation`](https://help.syncfusion.com/cr/xamarin/Syncfusion.SfNumericUpDown.XForms.SfNumericUpDown.html#Syncfusion_SfNumericUpDown_XForms_SfNumericUpDown_SpinValidation) is set to Auto.
