* \#I423570 - [Android] The [`IsEdited`](https://help.syncfusion.com/cr/xamarin/Syncfusion.SfImageEditor.XForms.SfImageEditor.html#Syncfusion_SfImageEditor_XForms_SfImageEditor_IsImageEdited) property will not be set to true when opening the Drawing or Shapes toolbar and tapping the Back key without placing any shapes.
