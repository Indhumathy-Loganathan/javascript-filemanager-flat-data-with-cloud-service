## SfPdfViewer

### Features
{:#sfpdfviewer-features}
* \#281178 - Support to display all selected symbols of radio button form fields has been implemented. 
* \#279824 - [Android] Improved the quality of the strokes of ink annotations.
[iOS, UWP] Support to display the text in signature form fields has been implemented.
