## SfNumericTextBox

### Bug fixes

* \#I362559 - [UWP] GroupSeparator will not be displayed after setting [`EnableGroupSeparator`](https://help.syncfusion.com/cr/xamarin/Syncfusion.SfNumericTextBox.XForms.SfNumericTextBox.html#Syncfusion_SfNumericTextBox_XForms_SfNumericTextBox_EnableGroupSeparator) as false when changing Culture.
