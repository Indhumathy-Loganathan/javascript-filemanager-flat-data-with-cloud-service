## SfComboBox

### Features
{:#sfcombobox-features}

* \#235635 - [`FontFamily`](https://help.syncfusion.com/cr/xamarin/Syncfusion.SfComboBox.XForms~Syncfusion.XForms.ComboBox.SfComboBox~FontFamily.html) support has been provided to Xamarin.Forms Standard.
* \#186723 - DataTemplateSelector support has been provided.
* \#221600 - [`SelectedIndex`](https://help.syncfusion.com/cr/xamarin/Syncfusion.SfComboBox.XForms~Syncfusion.XForms.ComboBox.SfComboBox~SelectedIndex.html) set access support has been provided.
* \#231439 - `DropDownBorderColor` support has been provided. 
* \#232655 - `SearchDelay` support has been provided.

### Bug Fixes
{:#sfcombobox-bug-fixes}

* \#236933 - [iOS] The SfCombobox is now rendered in a ListView ViewCell.
* \#235601 - [iOS] The SfComboBox text will not be changed when clicking the Done button in keyboard.

### Breaking Changes
{:#sfcombobox-api-breaking}

* \#235635 - [`TextSize`](https://help.syncfusion.com/cr/xamarin/Syncfusion.SfComboBox.XForms~Syncfusion.XForms.ComboBox.SfComboBox~TextSize.html) and [`DropDownTextSize`](https://help.syncfusion.com/cr/xamarin/Syncfusion.SfComboBox.XForms~Syncfusion.XForms.ComboBox.SfComboBox~DropDownTextSize.html) have been changed into double data type.
* [UWP] Watermark behavior has been changed as per the default Entry control.
