## SfChart

### Features
{:#sfchart-features}

* \#I205947 - Now, the chart control supports [`SFRangeColumnSeries`](https://help.syncfusion.com/cr/cref_files/xamarin-ios/Syncfusion.SFChart.iOS~Syncfusion.SfChart.iOS.SFRangeColumnSeries.html) chart type with a single y-value.
 
* \#I205947 - Now, the data label and [`PieSeries`](https://help.syncfusion.com/cr/cref_files/xamarin-ios/Syncfusion.SFChart.iOS~Syncfusion.SfChart.iOS.SFPieSeries.html)/[`DoughnutSeries`](https://help.syncfusion.com/cr/cref_files/xamarin-ios/Syncfusion.SFChart.iOS~Syncfusion.SfChart.iOS.SFDoughnutSeries.html) segments can be connected with horizontal straight line.
 
* \#I214334, I208403, I194500 - Now, the chart control supports series selection.
 
* \#F139960, I215900 - Now, the chart control supports custom axis.

### Enhancements
{:#sfchart-enhancements}
     
* \#F140540 - New legend icons such as [`Plus`](https://help.syncfusion.com/cr/cref_files/xamarin-ios/Syncfusion.SFChart.iOS~Syncfusion.SfChart.iOS.SFChartLegendIcon.html), [`Hexagon`](https://help.syncfusion.com/cr/cref_files/xamarin-ios/Syncfusion.SFChart.iOS~Syncfusion.SfChart.iOS.SFChartLegendIcon.html), and [`VerticalLine`](https://help.syncfusion.com/cr/cref_files/xamarin-ios/Syncfusion.SFChart.iOS~Syncfusion.SfChart.iOS.SFChartLegendIcon.html) have been added.
