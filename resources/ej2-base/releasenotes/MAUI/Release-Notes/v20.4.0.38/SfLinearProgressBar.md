## SfLinearProgressBar

### Features
{:#sflinearprogressbar-features}

* **Progress Padding** - Provided support to adjust the left and right edge padding of the progress and the secondary progress indicator in the [`SfLinearProgressBar`](https://help.syncfusion.com/cr/maui/Syncfusion.Maui.ProgressBar.SfLinearProgressBar.html#Syncfusion_Maui_ProgressBar_SfLinearProgressBar__ctor) using the [`ProgressPadding`](https://help.syncfusion.com/cr/maui/Syncfusion.Maui.ProgressBar.SfLinearProgressBar.html#Syncfusion_Maui_ProgressBar_SfLinearProgressBar_ProgressPadding) property.
