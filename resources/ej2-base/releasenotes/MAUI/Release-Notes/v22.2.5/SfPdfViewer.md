## SfPdfViewer `Preview`

### Features
{:#sfpdfviewer-features}

* Provided support to add, remove, and modify annotations in PDF files. The annotations can also be imported, exported, and saved as well. The following annotation types are now available.

  * Arrow
  * Circle
  * Highlight
  * Ink
  * Line
  * Square
  * Squiggly
  * Stamp
  * Strike-out
  * Underline
