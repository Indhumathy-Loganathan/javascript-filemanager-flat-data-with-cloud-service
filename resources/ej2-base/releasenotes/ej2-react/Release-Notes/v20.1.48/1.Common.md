## Common

### Features

- Fluent theme has been added with light and dark variants to all Syncfusion Angular UI components.

### Breaking Changes

- Starting with 2022 Volume 1 v20.1.0.47, all Syncfusion customers (evaluators and paid customers) who use Syncfusion installers or npm packages from npmjs must generate and register the corresponding platform and version license key in your projects. For more information, please see this [help topic](https://help.syncfusion.com/common/essential-studio/licensing/license-key)
- The un-versioned CDN links which always maintains latest version scripts are deprecated from 2022 Vol1 - 20.1 version. These links are available with `19.4` version scripts to avoid breaking in sites and apps that uses un-versioned links.
