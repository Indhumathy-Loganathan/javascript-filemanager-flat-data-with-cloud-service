## Tree Grid

### Features

- Provided the shimmer effect for tree grid while loading, refreshing and data processing. Please find the demo [here](https://ej2.syncfusion.com/aspnetmvc/TreeGrid/LoadingAnimation#/bootstrap5).
- `#I307202`, `#FB20655` - Provided support to load child records on demand while virtual scrolling in remote data binding. Find more details in the documentation [here](https://ej2.syncfusion.com/aspnetmvc/documentation/tree-grid/data-binding/remote-data#load-parent-rows-in-expanded-state-with-virtualization).
