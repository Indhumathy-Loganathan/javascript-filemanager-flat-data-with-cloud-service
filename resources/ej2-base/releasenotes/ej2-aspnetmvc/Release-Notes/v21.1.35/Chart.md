## Chart

### Features

- `#I320485` - Provided support to place a label at the center of the pie and donut charts. Find the UG documentation link [here](https://ej2.syncfusion.com/aspnetmvc/documentation/accumulation-chart/pie-dough-nut).
- `#I416444` - Provided support for a new chart type called range step area which is used to display the difference between minimum and maximum values over a certain time period. Find the demo link [here](https://ej2.syncfusion.com/aspnetmvc/Chart/RangeStepArea#/fluent).
- `#I396453`, `#I314160` - Provided support to customize the height and color of the error bar of each data point. Find the demo link [here](https://ej2.syncfusion.com/aspnetmvc/Chart/ErrorBar#/fluent).

### Breaking Changes

- Starting from version v21.1.x, the value type of `horizontalError`, `horizontalPositiveError`, `horizontalNegativeError`, `verticalError`, `verticalNegativeError` and `verticalPositiveError` properties in error bar has been changed from a primitive data type (double) to an object data type.
