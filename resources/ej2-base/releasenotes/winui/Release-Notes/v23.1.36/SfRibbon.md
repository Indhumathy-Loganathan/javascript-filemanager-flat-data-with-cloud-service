## SfRibbon `Preview`

### Features
{:#sfribbbon-feature}

* **Quick Access Toolbar (QAT)** - This feature allows users to render a set of ribbon items that are commonly used in applications at the top-left corner of a window to make them more accessible.

* **ScreenTip support** - Provided support to display a pop-up window when the pointer hovers over a ribbon item. It provides helpful details about the function of the ribbon item, making it easier for users to understand its purpose.
