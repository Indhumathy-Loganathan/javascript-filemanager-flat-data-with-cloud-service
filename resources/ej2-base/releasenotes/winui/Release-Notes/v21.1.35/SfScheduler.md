## SfScheduler

### Features
{:#sfscheduler-features}

* \#I436893 - Obtain recurrence user editing options (single or series selection for recurring appointments) in [AppointmentEditorOpening](https://help.syncfusion.com/cr/wpf/Syncfusion.UI.Xaml.Scheduler.SfScheduler.html#Syncfusion_UI_Xaml_Scheduler_SfScheduler_AppointmentEditorOpening) and [AppointmentDeleting](https://help.syncfusion.com/cr/wpf/Syncfusion.UI.Xaml.Scheduler.SfScheduler.html#Syncfusion_UI_Xaml_Scheduler_SfScheduler_AppointmentDeleting) events.
* The default appointment rendering has been improved based on the available size of the cell in month view.
* Now, the [`Scheduler`](https://help.syncfusion.com/cr/winui/Syncfusion.UI.Xaml.Scheduler.SfScheduler.html) appointments have mouse hovering support.
