## SfPyramidChart `Preview`

### Breaking Changes
{:#chart-breaking-changes}

* The selection feature's APIs revamped based on the application usage. For current API details in the selection feature, refer to this [link](https://help.syncfusion.com/winui/pyramid-chart/selection).
* `ShowTooltip` property name changed to `EnableTooltip` in `SfPyramidChart`. 
* `ChartLegend` base class changed from `ItemsControl` to `DependencyObject`.
* The `LegendItem` class accessibility changed from public to internal.
