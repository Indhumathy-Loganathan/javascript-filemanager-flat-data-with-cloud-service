## RadialGauge

### Features
{:#sfradialgauge-features}

* **Shadow support** - Provided shadow support for [`RadialGauge`](https://help.syncfusion.com/cr/winui/Syncfusion.UI.Xaml.Gauges.SfRadialGauge.html)'s [`ShapePointer`](https://help.syncfusion.com/cr/winui/Syncfusion.UI.Xaml.Gauges.ShapePointer.html) by introducing `HasShadow` property.
* **Overlay support** - Provided overlay support for [`RadialGauge`](https://help.syncfusion.com/cr/winui/Syncfusion.UI.Xaml.Gauges.SfRadialGauge.html)'s [`ShapePointer`](https://help.syncfusion.com/cr/winui/Syncfusion.UI.Xaml.Gauges.ShapePointer.html) along with `OverlayFill` and `OverlayRadius` customizations.
