## SfDataGrid

### Bug fixes
{:#sfdatagrid-bug-fixes}

* \#I446133 - The `ArgumentOutOfRangeException` will no longer be thrown when clicking the AddNewRow icon with DataGrid's [NavigationMode](https://help.syncfusion.com/cr/winui/Syncfusion.UI.Xaml.Grids.SfGridBase.html#Syncfusion_UI_Xaml_Grids_SfGridBase_NavigationMode) as [Row](https://help.syncfusion.com/cr/winui/Syncfusion.UI.Xaml.Grids.NavigationMode.html#Syncfusion_UI_Xaml_Grids_NavigationMode_Row).

* \#I445067 - Now, when entering edit mode by pressing a number key, the first number entered will not be overwritten by subsequent numbers in [GridNumericColumn](https://help.syncfusion.com/cr/winui/Syncfusion.UI.Xaml.DataGrid.GridNumericColumn.html).
