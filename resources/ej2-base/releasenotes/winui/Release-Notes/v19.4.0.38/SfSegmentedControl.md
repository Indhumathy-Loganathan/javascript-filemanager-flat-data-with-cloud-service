## SfSegmentedControl `Preview`

### Features

* **Shadow effect** - This feature provides options to enable a shadow effect for selected segment items.

### Breaking Changes

* Previously we can customize the appearance of selected item using the [SelectedSegmentStyle](https://help.syncfusion.com/cr/winui/Syncfusion.UI.Xaml.Editors.SfSegmentedControl.html#Syncfusion_UI_Xaml_Editors_SfSegmentedControl_SelectedSegmentStyle) property which accepts style of target type [Border](https://docs.microsoft.com/en-us/windows/winui/api/microsoft.ui.xaml.controls.border) but we have changed the target type as [SelectedSegmentBorder](https://help.syncfusion.com/cr/winui/Syncfusion.UI.Xaml.Editors.SelectedSegmentBorder.html) in order to accomodate more customization settings for selected item.
