## DatePicker

### Features
{:#sfdatepicker-features}

* Provided support to show calendar fields in different calendar types.

* Provided [`DateChanging`](https://help.syncfusion.com/winui/date-picker/getting-started#cancel-a-date-that-is-being-changed) event support to cancel a date that is being changed.

* Now date value entered in editor when [`EditMode`](https://help.syncfusion.com/winui/date-picker/getting-started#edit-date-using-free-form-editing) is Mask, is updated only when focus is lost or Enter key press.
