## TimePicker 

### Features
{:#sftimepicker-features}

* Provided support to show time fields in twelve hour or twenty four hour types.

* Provided [`TimeChanging`](https://help.syncfusion.com/winui/time-picker/getting-started#cancel-a-time-that-is-being-changed) event support to cancel time that is being changed.

* Now time value entered in editor when [`EditMode`](https://help.syncfusion.com/winui/time-picker/getting-started#edit-time-using-free-form-editing) is Mask, is updated only when focus is lost or Enter key press.
