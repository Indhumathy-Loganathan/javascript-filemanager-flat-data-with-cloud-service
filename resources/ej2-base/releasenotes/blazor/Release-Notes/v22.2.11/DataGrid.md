## Data Grid

### Bug Fixes

- `#I488320` - Resolved the issue where empty space appears on the right side of the grid toolbar when the [sticky header](https://blazor.syncfusion.com/documentation/datagrid/scrolling#sticky-header) is enabled.
- `#F184065` - Resolved the issue where a NullReferenceException was thrown when [filtering](https://blazor.syncfusion.com/documentation/datagrid/filtering) or [searching](https://blazor.syncfusion.com/documentation/datagrid/searching) with empty column values in an [ExpandoObject](https://blazor.syncfusion.com/documentation/datagrid/data-binding#expandoobject-binding).
- `#FB45405` - Resolved the issue where the command button triggered the [EndEdit](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_EndEditAsync) method twice upon pressing the enter key.
