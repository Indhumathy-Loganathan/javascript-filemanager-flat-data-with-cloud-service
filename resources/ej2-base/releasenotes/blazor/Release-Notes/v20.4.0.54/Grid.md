## Grid

### Bug Fixes

- `#I439102` - [Searching](https://blazor.syncfusion.com/documentation/datagrid/searching) is not working properly when grid is binded with [expando object](https://blazor.syncfusion.com/documentation/datagrid/data-binding#expandoobject-binding) and while a grid column is having empty values has been resolved. 
- `#I439670` - Focus is not proper while saving the edited record through Tab/ShiftTab key, when persist [selection](https://blazor.syncfusion.com/documentation/datagrid/selection) is enabled in the DataGrid is fixed.
- `#I442232` - Issue with "Missing "Pager_Items" key in Resource in v20.4.0.51" has been resolved.
- `#I436948` - Issue with "[Aggregate](https://blazor.syncfusion.com/documentation/datagrid/aggregates) value is not proper, while [exporting to PDF](https://blazor.syncfusion.com/documentation/datagrid/pdf-export)" has been resolved.
- `#I441377` - Issue with "When [AllowRowDragAndDrop](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_AllowRowDragAndDrop) is enabled within the grid, the focus is being applied to the indent cell instead of the data cell while saving the edited data" has been fixed.
- `#I441222` - Issue with "[Scrolling](https://blazor.syncfusion.com/documentation/datagrid/scrolling) is not proper with less number of records, while editing and [row virtualization](https://blazor.syncfusion.com/documentation/datagrid/virtualization#row-virtualization) is enabled” has been resolved.
- `#I438761` - Issue with “[RowDeselectedEvent](https://blazor.syncfusion.com/documentation/datagrid/events#rowdeselected) does not triggers properly, when freeze line is moved” is fixed.
- `#I441244` - Issue with “ReadAsync is invoked twice, when we navigate the page using pager template with [Custom DataAdaptor](https://blazor.syncfusion.com/documentation/data/custom-binding)” is resolved.
- `#I439936` - Issue with "[GetBatchChangesAsync](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_GetBatchChangesAsync) methods return value is not proper, when edit mode is set to batch" has been resolved.
- `#I437250` - Issue with “Horizontal scrollbar isn't rendering properly in [Frozen columns](https://blazor.syncfusion.com/documentation/datagrid/scrolling#frozen-rows-and-columns) on splitter resizing” is fixed.
- `#I435249` - “Empty space and alignment issues occur during horizontal scrolling in Frozen with [ColumnVirtualization](https://blazor.syncfusion.com/documentation/datagrid/virtualization#column-virtualization)“ has been resolved.
