##  Card

###    Breaking Changes

- The `ImageSrc` property of `CardHeader` has been changed to `ImageUrl`.

**Previous**

```csharp
    <SfCard>
            <CardHeader ImageSrc="images/cards/steven.png" />                
    </SfCard>
```

**Now**

```csharp
      <SfCard>
            <CardHeader ImageUrl="images/cards/steven.png" />                
      </SfCard>
```
