## Scheduler

### New Features

- Provided option to render the spanned events greater than 24 hours in either `AllDayRow` region or `TimeSlot` region.
- Provided localization support for timezone collections which are displayed in the editor window.

### Bug Fixes

- `#F170021` - An issue with the exception throws when we set `ShowQuickInfo` property as false has been fixed.
- `#I347737` - An issue with drag and drop is not working for recurrence event at initial time has been fixed.
