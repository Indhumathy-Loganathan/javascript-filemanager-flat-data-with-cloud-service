## MultiSelect Dropdown

### Breaking Changes

- The `E` property in [MultiSelectChangeEventArgs&lt;T&gt;](hhttps://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.DropDowns.MultiSelectChangeEventArgs-1.html) , [RemoveEventArgs&lt;T&gt;](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.DropDowns.RemoveEventArgs-1.html) , [TaggingEventArgs&lt;T&gt;](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.DropDowns.TaggingEventArgs-1.html) classes has been deprecated and will no longer be used. It will be removed in a future version.

- The `Element` property in [MultiSelectChangeEventArgs&lt;T&gt;](hhttps://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.DropDowns.MultiSelectChangeEventArgs-1.html) class has been deprecated and will no longer be used. It will be removed in a future version.

- The `Item` property in  [RemoveEventArgs&lt;T&gt;](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.DropDowns.RemoveEventArgs-1.html) class has been deprecated and will no longer be used. It will be removed in a future version.

- The `Event` and `Items` property in [SelectAllEventArgs&lt;T&gt;](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.DropDowns.SelectAllEventArgs-1.html) class has been deprecated and will no longer be used. It will be removed in a future version.

### Bug Fixes 

- `#I431010` - Issue with "the keypad not opening only on iOS devices when focusing the input" has been resolved.
