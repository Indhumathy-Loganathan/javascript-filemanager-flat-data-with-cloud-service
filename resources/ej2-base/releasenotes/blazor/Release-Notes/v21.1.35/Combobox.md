## ComboBox

### Features

- In virtualization functionality, support for increasing and decreasing the 'Skip' parameter based on scrolling direction has been added. Previously, the Take parameter increased while the Skip parameter had no effect on scrolling. Find the UG documentation link [here](https://blazor.syncfusion.com/documentation/combobox/virtualization).

### Bug Fixes

- `#SF-445918` - The issue with "Combobox component Focusin and Focusout the Empty value, Initial value was updated" has been fixed.
