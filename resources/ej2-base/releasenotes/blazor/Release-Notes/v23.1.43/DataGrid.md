## Data Grid

### Bug Fixes

- `#I508449` - Resolved the issue related to [filtering](https://blazor.syncfusion.com/documentation/datagrid/filtering) [complex columns](https://blazor.syncfusion.com/documentation/datagrid/columns#complex-data-binding) while binding data source from SQL server.
- `#I514892` - Resolved the issue where a NullReferenceException occurred when [exporting to excel](https://blazor.syncfusion.com/documentation/datagrid/excel-exporting), while the grid was bound with [DataManager](https://blazor.syncfusion.com/documentation/data/getting-started) and [aggregates](https://blazor.syncfusion.com/documentation/datagrid/aggregates).
- `#FB47531` - Resolved the issue where the [search](https://blazor.syncfusion.com/documentation/datagrid/searching) operation was not working properly when [columns](https://blazor.syncfusion.com/documentation/datagrid/columns) were hidden in the DataGrid.
