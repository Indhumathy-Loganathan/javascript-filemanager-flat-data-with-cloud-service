## Grid

### Features

 - `#F168527` -  Provided event support for [Detail Template](https://blazor.syncfusion.com/documentation/datagrid/detail-template) while expanding and collapsing.

 **Events Provided**

Event name | Description 
 --- | ---  
DetailsExpanding | An Event that is raised when [Detail template](https://blazor.syncfusion.com/documentation/datagrid/detail-template) row is clicked to expand, you can also cancel this action by using Cancel property provided in the event args.
DetailsExpanded | An Event that is raised after [Detail Template](https://blazor.syncfusion.com/documentation/datagrid/detail-template) row is expanded.
DetailsCollapsing | An event that is raised when [Detail Template](https://blazor.syncfusion.com/documentation/datagrid/detail-template) row is clicked to collapse,  you can also cancel this action by using Cancel property provided in the event args.
DetailsCollapsed | An event that is raised after [Detail Template](https://blazor.syncfusion.com/documentation/datagrid/detail-template) row is collapsed.
