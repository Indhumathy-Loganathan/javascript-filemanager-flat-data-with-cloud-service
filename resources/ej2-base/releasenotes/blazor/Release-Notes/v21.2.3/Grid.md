## Grid

### Bug Fixes

- `#I457538` - Exception throws while [selecting](https://blazor.syncfusion.com/documentation/datagrid/selection) the rows using Shift+Ctrl keys has been resolved.
- `#F181510` - Exception throws while copying the cells after [scrolling](https://blazor.syncfusion.com/documentation/datagrid/scrolling) the grid, when [virtualization](https://blazor.syncfusion.com/documentation/datagrid/virtualization) is enabled is fixed.
- `#I456600` - Issue with "DataGrid Virtual Column's is not rendering properly with [Expando Object](https://blazor.syncfusion.com/documentation/datagrid/data-binding#expandoobject-binding), when using the [column chooser](https://blazor.syncfusion.com/documentation/datagrid/column-chooser) in WASM application" is fixed.
- `#I453773` - Issue with "Tab and ShiftTab keys doesn't works as expected, when the [CheckboxMode](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.GridSelectionSettings.html#Syncfusion_Blazor_Grids_GridSelectionSettings_CheckboxMode) is set to [ResetOnRowClick](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.CheckboxSelectionType.html#Syncfusion_Blazor_Grids_CheckboxSelectionType_ResetOnRowClick) while using the [persist selection](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.GridSelectionSettings.html#Syncfusion_Blazor_Grids_GridSelectionSettings_PersistSelection)" has been resolved.
- `#F181659` - Issue with "Hiding the column header using [HideAtMedia](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.GridColumn.html#Syncfusion_Blazor_Grids_GridColumn_HideAtMedia) property doesn't work properly" is fixed.
- `#I436130` - Issue with "Grid header scrollbar padding is not removed, while deleting the records in the current view until the grid content scrollbar is getting hidden" has been resolved.
- `#F181648` – Issue with “vertical text alignment and textwrap is not applied in [excel export](https://blazor.syncfusion.com/documentation/datagrid/excel-exporting)” is fixed.
- `#I453433` - Issue with "Grid [ResetPersistDataAsync](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_ResetPersistDataAsync) doesn't restore default grid [sorting](https://blazor.syncfusion.com/documentation/datagrid/sorting)" has been resolved.
- `#I452378` - Issue with "Tooltip of Grid with [ClipMode](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_ClipMode) of type [EllipsisWithTooltip](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.ClipMode.html#Syncfusion_Blazor_Grids_ClipMode_EllipsisWithTooltip) stays visible after [filtering](https://blazor.syncfusion.com/documentation/datagrid/filtering)" has been resolved.
