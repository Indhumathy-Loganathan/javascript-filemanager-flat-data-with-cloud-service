## Grid

### Bug Fixes

- `#I432017` - Issue with "Grid vertical scrolling not working work in [virtualization](https://blazor.syncfusion.com/documentation/datagrid/virtualization), when [grouping](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_AllowGrouping) is enabled" is resolved.
- `#I433068`, `#F180327`, `#I438346` - Issue with "Column header and content is not aligned properly" is resolved.
- `#I434769` - Issue with "Records not rendering properly, when using [RowHeight](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_RowHeight) with [virtualization](https://blazor.syncfusion.com/documentation/datagrid/virtualization)" has been resolved.
- `#I434479`  - Issue with "Null Reference Exception caused by a [Checkbox filter](https://blazor.syncfusion.com/documentation/datagrid/check-box-filter) on a nullable property in combination with Format during [filtering](https://blazor.syncfusion.com/documentation/datagrid/filtering)" has been resolved.
- `#I436708` - “Null Reference Exception throws in [custom Validation](https://blazor.syncfusion.com/documentation/datagrid/column-validation#custom-validation), while displaying the [built-in validation message tooltip](https://blazor.syncfusion.com/documentation/datagrid/column-validation#display-validation-message-using-in-built-tooltip) when a record is saved" has been resolved.
- `#I434715` - Exception thrown when [filtering](https://blazor.syncfusion.com/documentation/datagrid/filtering) a complex column, using [ODataV4Adaptor](https://blazor.syncfusion.com/documentation/datagrid/data-binding#binding-with-odata-v4-services) has been resolved.
- `#F180159` - Exception throws on performing [row drag and drop](https://blazor.syncfusion.com/documentation/datagrid/row-drag-and-drop) between grid to other components" has been resolved.
- `#FB37613` - Issue with "When dynamically changing [IsFrozen]((https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.GridColumn.html#Syncfusion_Blazor_Grids_GridColumn_IsFrozen)) property, frozen and movable table doesn't rendered properly" has been resolved.
