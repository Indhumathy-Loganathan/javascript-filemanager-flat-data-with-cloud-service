## Grid

### Features

- `#I287323` , `#F178937` - Provided row [drag and drop](https://blazor.syncfusion.com/documentation/datagrid/row-drag-and-drop) support for dragging and dropping between two different model grids.

### Bug Fixes

- `#I415497` - Issue with "[Drag and Drop](https://blazor.syncfusion.com/documentation/datagrid/row-drag-and-drop) between the grids is not working properly in the firefox browser" has been resolved.
- `#I430782` - In [Dialog Edit](https://blazor.syncfusion.com/documentation/datagrid/dialog-editing), the dialog doesn't render properly when the EnableAdaptiveUI is set to true is fixed.
- `#I432593` - Issue with "[SelectedRowIndex](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_SelectedRowIndex) property doesn't get updated during [selection](https://blazor.syncfusion.com/documentation/datagrid/selection)" has been resolved.
- `#F180203` - Issue with "In [Adaptive](https://blazor.syncfusion.com/documentation/datagrid/adaptive-layout) mode, the filter dialog displays column name instead of display name" has been resolved.
- `F177937` - Issue with "[RowDragStarting](https://blazor.syncfusion.com/documentation/datagrid/events#rowdragstarting) event doesn't triggers, while performing the [Drag and Drop](https://blazor.syncfusion.com/documentation/datagrid/row-drag-and-drop) action" is fixed.
- `#I419734` - Exception throws while adding or removing the [frozen columns](https://blazor.syncfusion.com/documentation/datagrid/scrolling#frozen-rows-and-columns) dynamcially is fixed.
- `#I429558` - Issue with "[Validation message](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.GridColumn.html#Syncfusion_Blazor_Grids_GridColumn_ValidationRules) not visible properly when Grid has empty data" has been resolved.
