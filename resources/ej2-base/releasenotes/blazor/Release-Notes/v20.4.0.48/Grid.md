## Grid

### Bug Fixes

- `#I418627` - The issue with console errors being thrown when trying to add or edit a record in a grid containing multiple [GridForeignColumn](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.GridForeignColumn-1.html) with the same field names and conflicting [validation rules](https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.GridColumn.html#Syncfusion_Blazor_Grids_GridColumn_ValidationRules) has been resolved.
- `#I431382` - The issue with console errors being thrown while selecting a row in the datagrid that consists of a [detail template](https://blazor.syncfusion.com/documentation/datagrid/detail-template) has been resolved.
- `#I429646` - The issue with missing translation resource keys in v20.4.0.43 has been resolved.
