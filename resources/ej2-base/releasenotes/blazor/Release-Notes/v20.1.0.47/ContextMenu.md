## Context Menu

### Breaking Changes

**Properties**

The following property names in `SfContextMenu` have been changed,

|Component|Old Property Name |Old Property Type |New Property Name|New Property Type | Comments |
|-------------|------------|------------|------------|------------|------------|
|SfContextMenu|`ShowOn`|string |`OpenActionEvents`| string | Old property is deprecated. |
|SfContextMenu|`CloseOn`|enum |`CloseActionEvents`| string | Old property is removed, since it was introduced only in 19.4.53 weekly NuGet release |
