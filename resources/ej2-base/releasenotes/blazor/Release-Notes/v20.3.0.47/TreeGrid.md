## Tree Grid

### Features

- `#I374356` - Provided support for state persistence on browser reload. Tree grid state can also be saved and restored manually. Please find the documentation link [here](https://blazor.syncfusion.com/documentation/treegrid/state-management).
- Users can add or remove frozen columns by dragging the column separator line. Please find the demo link [here](https://blazor.syncfusion.com/demos/tree-grid/frozen-column?theme=fluent).
