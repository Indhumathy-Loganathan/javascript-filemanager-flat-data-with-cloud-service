### Features
{:#sfchart-features}

* The stacked doughnut support has been provided to chart.

* Now, the chart control supports [`RangeColumnSeries`](https://help.syncfusion.com/cr/cref_files/uwp/Syncfusion.SfChart.UWP~Syncfusion.UI.Xaml.Charts.RangeColumnSeries.html) chart type with a single y-value.

* Now, the data label and [`PieSeries`](https://help.syncfusion.com/cr/cref_files/uwp/Syncfusion.SfChart.UWP~Syncfusion.UI.Xaml.Charts.PieSeries.html)/[`DoughnutSeries`](https://help.syncfusion.com/cr/cref_files/uwp/Syncfusion.SfChart.UWP~Syncfusion.UI.Xaml.Charts.DoughnutSeries.html) segments can be connected with horizontal straight line.

* Support has been provided to group the data points into `Others` category in [`PieSeries`](https://help.syncfusion.com/cr/cref_files/uwp/Syncfusion.SfChart.UWP~Syncfusion.UI.Xaml.Charts.PieSeries.html) and [`DoughnutSeries`](https://help.syncfusion.com/cr/cref_files/uwp/Syncfusion.SfChart.UWP~Syncfusion.UI.Xaml.Charts.DoughnutSeries.html) charts.

* The text or any view can be added to the center of the doughnut chart.

### Enhancements
{:#sfchart-enhancements}

* Now, the dashes can be applied to the polar and radar charts when rendering as lines.
