## SfChart

### Bug fixes
{:#sfchart-bug-fixes}

*  \#217110  - Now, the System.Exception will no longer be thrown when charts are generated in multiple pages.

*  \#217110  - Now, [`FastRangeAreaBitmapSeries`](https://help.syncfusion.com/cr/cref_files/uwp/Syncfusion.SfChart.UWP~Syncfusion.UI.Xaml.Charts.FastRangeAreaBitmapSeries.html) and [`FastLineBitmapSeries`](https://help.syncfusion.com/cr/cref_files/uwp/Syncfusion.SfChart.UWP~Syncfusion.UI.Xaml.Charts.FastLineBitmapSeries.html) will no longer be drawn beyond the rows and columns definition when zooming.

### Breaking changes	
{:#sfchart-breaking-changes}

*  \#217509 - Now, the selection zooming works based on rows and columns definition of the chart.
