## Visual Studio Extensions

### Breaking changes

* The ASP.NET Core project template is no longer supported for Framework versions that are no longer supported, such as.NET Core 3.0, 2.2, and 2.1, as well as the Visual Studio 2017.
* The ASP.NET Core sample creator is no longer supported for Framework versions that are no longer supported, such as.NET Core 3.0, 2.2, and 2.1, as well as the Visual Studio 2017.

### New Feature

* Provided the .NET 6.0 support in ASP.NET Core Scaffolding command line.
